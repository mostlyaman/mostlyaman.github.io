CSS Cascading Style Sheets (CSS) serves as the design language of the web, allowing you to control the visual appearance and layout of HTML elements. This cheatsheet will introduce you to some of the most basic features of CSS. Again, you dont have to remember all of the CSS properties at once. You will gradually learn them with experience.

This cheatsheet is based on the free <a href="https://youtu.be/yfoY53QXEnI" target="__blank">CSS Crash Course by Traversy Media</a>.

### Adding CSS to HTML

There are three ways to use CSS in a HTML file: using the `style attribute`, or by using the `\<style> tag`, or by including a `external css file`. You can try them in the editor below. 

#### Method 1: style attribute

```html
<p
  style="background-color: red; color: white;"
>This is a paragraph</p>
```

<Playground />

#### Method 2: style tag

<div class="h-4"></div>
```html
<p>This is a paragraph.</p>
<style>
  p {
    background-color: red;
    color: white;
  }
</style>
```

<Playground />

#### Method 3: External CSS File

<div>HTML Code:</div>
```html
<p>This is a paragraph.</p>
```

<div>CSS Code:</div> 
```css

  background-color: red;
  color: white;
}
```

### Font Properties

<Remember text="font-family property can be used to change the font of the text in a HTML element. You need to import your fonts, if they are not installed already on the user's device." />

Read about Web Safe fonts <a href="https://www.w3schools.com/cssref/css_websafe_fonts.php" target="__blank">here</a>.

<Remember text="font-size property can be used to change the size of the text." />

<Remember text="font-weight property can be used to change the boldness of the text." />

<div>HTML Code</div>
```html
<p>
CSS font properties offer versatile options to style text on web pages. 
With properties like font-family, font-size, font-weight, and font-style,
designers can customize typography to match the desired aesthetic and 
enhance readability. Additionally, properties such as line-height and
letter-spacing allow for fine-tuning text presentation for optimal 
user experience.
</p>
```
<div class="h-4"></div>

<div>CSS Code</div>
```css
p {
  font-family: Monospace, sans-serif;
  font-size: 24px;
  font-weight: bold;
}
```
