### Document Object Model

Within the browser context, Javscript has access to the `document` object which can be used to interact with the HTML. All the HTML elements on the page are expressed as a tree. You can add, remove or change the attributes of any element using the `document` object.

<img class="mx-auto" src="/images/post-6/dom.png" alt="DOM example" />

<Remember text="document.body contains all the HTML elements in the webpage in a tree-like data structure." />

```html
<h1>Heading 1</h1>
<p>This is a paragraph.</p>
```
<div class="h-4"></div>
```js
const elements = document.body.children

for(let i = 0; i<elements.length; i++) {
  console.log(elements[i].innerText)
}
```

### Reference by ID

You can reference a single element by their id. 

<Remember text="Use getElementById() to search the DOM tree for specific element with a certain id you would like to interact with. Note that this return reference to the first element with the matching id." />

```html
<div style="display:flex; flex-direction: column; gap: 3px;
    color: white">
  <div style="background: black; padding: 20px;" id="counter-value">0</div>
</div>
<button onclick="increment()" style="padding: 5px; margin: 20px;">
  Increment Counter
</button>
```
<div class="h-4"></div>
```js
function increment () {
  const counter = document.getElementById('counter-value')
  const value = parseInt(counter.innerText)
  counter.innerText = (value+1).toString()
}
```

### Reference by Class Name

You can reference all the elements in the webpage having a certain class name.

<Remember text="Use getElementsByClassName() to search the DOM tree for specific element's' with a certain class name you would like to interact with. Note that this returns a list containing all the elements with the matching class name." />

```html
<div style="display:flex; flex-direction: column; gap: 3px;
    color: white">
  <div style="background: black; padding: 20px;" class="counter-value">0</div>
  <div style="background: black; padding: 20px;" class="counter-value">0</div>
</div>
<button onclick="increment()" style="padding: 5px; margin: 20px;">
  Increment Counter
</button>
```
<div class="h-4"></div>
```js
function increment () {
  const counters = document.getElementsByClassName('counter-value')

  for(let i=0; i<counters.length; i++) {
    let counter = counters[i]
    const value = parseInt(counter.innerText)
    counter.innerText = (value+1).toString()
  }
}
```

### Change styling of a HTML element

<Remember text="You can change the styling of a HTML element using the style property." />

```html
<div id="box" style="height: 20px; width: 20px margin: 20px"></div>
<input type="text" id="color-input" placeholder="Enter Colour Name" />
<button onclick="changeColour()"></button>
```
<div class="h-4"></div>
```js
function changeColour() {
  const input = document.getElementById('color-input')
  const colorName = input.value

  const box = document.getElementById('box')
  box.style.backgroundColor = colorName
}

```
